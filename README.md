# Dr. Mario — Assembly Project

Assembly language game project inspired by *Dr. Mario*.

## Overview
A falling-capsule puzzle game where the player clears viruses by matching **four of the same color in a row**. The game includes gravity, increasing speed, sound effects, and multiple difficulty levels.

## Features
- Keyboard-controlled capsule movement and rotation  
- Automatic falling with increasing speed  
- Match-4 elimination mechanics  
- Easy / Medium / Hard difficulty selection  
- Pause functionality  
- Game over and win conditions  
- Sound effects and bitmap graphics (Dr. Mario, viruses)

## How to Play
- Press **1 / 2 / 3** to select difficulty  
- Controls:
  - **A**: move left  
  - **D**: move right  
  - **S**: move down  
  - **W**: rotate capsule (90° clockwise)  
- Eliminate all viruses to win  
- Game ends if capsules reach the top of the bottle

## Display Settings
- Width: 80  
- Height: 80  
- Unit size: 2
